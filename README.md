# 猫画像ジェネレーター

**目的：Next.js（TypeScript）の学習のため**　<br />
<br />
ボタンをクリックすると、APIから画像のURLを取得し、ランダムに可愛い猫画像を表示する。

[![Image from Gyazo](https://i.gyazo.com/58bfc5507b6c1fef1f69db0d02f5eb79.gif)](https://gyazo.com/58bfc5507b6c1fef1f69db0d02f5eb79)

# 使用技術
- TypeScript 5.0.4
- Next.js 13.4.2
- Jest 29.5.0（テスト）
  
# プロジェクトの起動

```
npm run dev
```
